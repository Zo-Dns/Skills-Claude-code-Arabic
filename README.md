# 🚀 Skills Claude Code - Arabic

مجموعة مهارات مخصصة وسكربتات ذكية لأداة Claude Code، مصممة خصيصاً لأتمتة وتوليد المناهج والدروس التقنية لمنصة JeelPixel التعليمية الموجهة للأطفال والمراهقين (من 8 إلى 16 سنة). 

تتميز هذه المهارات بقدرتها على صياغة المحتوى باللغة العربية الفصحى (بلا تشكيل)، ومواءمة العمق المعرفي حسب الفئات العمرية، وتوليد مخرجات برمجية جاهزة بصيغة TypeScript لإدراجها مباشرة في ملف الاستزراع `seed-lessons.ts`.

---

## 🛠️ التخصصات الستة المدعومة (The 6 Core Skills)

يحتوي هذا المستودع على 6 مهارات أساسية تغطي المحاور التقنية للمنصة:

### 1. 🤖 قسم الذكاء الاصطناعي (`ai-curriculum`)
- الشخصية الإرشادية: د. عمر (باحث أول في الذكاء الاصطناعي).
- الوظيفة: توليد دروس تفاعلية وقرائية تغطي مفاهيم تعلم الآلة (ML)، الشبكات العصبية، تصنيف الصور، ومعالجة اللغات الطبيعية دون تعقيدات رياضية ثقيلة.

### 2. 🌱 قسم الأساسيات التقنية (`cs-fundamentals`)
- الشخصية الإرشادية: د. خالد (أستاذ علم الحاسوب وأنظمة التشغيل).
- الوظيفة: تبسيط المفاهيم العميقة لبنية الحاسوب (مثل المعالج CPU، الذاكرة RAM، نظام التشغيل، والأنظمة الثنائية) باستخدام تشبيهات قريبة من واقع الطفل وعقليته.

### 3. 🛡️ قسم الأمن السيبراني والدفاع (`cybersecurity-defense`)
- الشخصية الإرشادية: د. ليلى (خبيرة أمن سيبراني ومديرة أمن معلومات CISO).
- الوظيفة: بناء الوعي الدفاعي والحماية الرقمية وشرح مبادئ (CIA Triad)، التشفير الآمن، والوقاية من الهندسة الاجتماعية والبرمجيات الخبيثة.

### 4. 🥷 قسم الهاكر الأخلاقي (`ethical-hacking`)
- الشخصية الإرشادية: Ghost (باحث أمني ومختبر اختراق أقدم).
- الوظيفة: غرس عقلية البحث الاستكشافية وتعليم أدوات الاختراق الأخلاقي (Nmap, Burp Suite, OSINT) في بيئات تدريبية معزولة وضمن إطار قانوني وأخلاقي صارم وممنهج.

### 5. 🌐 قسم الشبكات وعالم الإنترنت (`networking-expert`)
- الشخصية الإرشادية: مهندس أحمد (مهندس معمارية شبكات أقدم).
- الوظيفة: تتبع رحلة حزم البيانات (Packets) وشرح البنية التحتية للإنترنت، العناوين (IP/DNS)، بروتوكولات الويب (HTTP/HTTPS)، والـ VPN بأسلوب تصوري مبسط.

### 6. 💻 قسم إتقان البرمجة (`programming-mastery`)
- الشخصية الإرشادية: المهندسة سارة (مهندسة برمجيات أقدم وخبير Clean Code).
- الوظيفة: توليد تحديات التفكير المنطقي والبرمجي، وكتابة شيفرات برمجية حقيقية بلغة Python و JavaScript قابلة للتنفيذ المباشر داخل المتصفح مع حساب التعقيد الزمني تصوّرياً.

## ⚙️ آلية العمل والاستدعاء (Usage)

عند تشغيل Claude Code داخل بيئة التطوير، يمكنك استدعاء هذه المهارات تلقائياً عبر كتابة أوامر برمجية أو طلبات طبيعية مثل:
- `"أنشئ درساً في قسم الذكاء الاصطناعي عن تصنيف الصور"` -> يقوم باستدعاء سكيل `ai-curriculum`.
- `"صمم تحدي برمجياً لدرس بايثون للمبتدئين"` -> يقوم باستدعاء سكيل `programming-mastery`.

## 📌 القواعد العامة للمخرجات
تلتزم جميع المهارات بضوابط موحدة محددة في ملف الـ `CLAUDE.md`:
- مخرجات بصيغة TypeScript نظيفة وجاهزة للصق دون تداخل مع قواعد البيانات (No SQL).
- الالتزام التام بمفاتيح التفاعل المدعومة لكل كورس (`interactive` أو `reading` أو `coding`).
- التوازن الدقيق بين تبسيط المعلومة والمحافظة على دقتها العلمية والتاريخية دون تضليل.
